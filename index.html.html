О системе DOPHAMINE 90

Флагманская алюминиевая система с полностью скрытой створкой. Профиль шириной 90 мм с терморазрывом PA66GF25 (34 мм) обеспечивает выдающиеся показатели тепло- и звукоизоляции. Благодаря скрытому монтажу створки достигается минималистичный архитектурный облик — снаружи видно только стекло. Идеальное решение для современного дизайна фасада.

Премиальная подъёмно-сдвижная алюминиевая система для панорамного остекления. Профиль шириной 120 мм с терморазрывом PA66GF25 (14,8 мм) сочетает выдающуюся жёсткость конструкции с отличными теплоизоляционными свойствами.
